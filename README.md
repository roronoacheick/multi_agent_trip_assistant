# Multi-Agent Trip Assistant

Projet d'assistant multi-agents en Python (console) qui :
- analyse un message utilisateur libre en français,
- vérifie la météo réelle via OpenWeather,
- propose des idées d'activités et de logements (logements simulés),
- construit 2–3 scénarios de sorties dans un budget donné.

Technos principales :
- Python 3.12
- Groq (LLM) pour l'analyse de la demande et la présentation
- OpenWeather API pour la météo réelle
